Loops are things supposed to happen again and again.
1. for loop
fruits = ["Apple", "Peach", "Pear"]
for fruit in fruits:
  print(fruit)

2. Using the range function
for number in range(a, b):
  print(number)
   # all numbers between a and b and not including b

for number in range(a, b, c)
  print(number)
    # from a the +c to gives the next number. The default value is 1.
