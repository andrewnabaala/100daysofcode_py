Random Module.
Python uses a pseudo random number generator called Mersenne Twister (https://en.wikipedia.org/wiki/Mersenne_Twister)
    Khan Academy has made easier for you to understand the number generators (https://www.khanacademy.org/computing/computer-science/cryptography/crypt/v/random-vs-pseudorandom-number-generators)
But we won't implement the Mersenne Twister ourselves. So we use random modules, import random modules first then you will be able to use the functions in the module.
    eg: randint(a, b) will generate a random integer between a and b(both inclusive)
